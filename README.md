## AluraFlix-Online

Proyecto hecho con React sobre una pagina con videos educativos

## Instalacion

Clona este repositorio. Necesitará `node` y `npm` instalados globalmente en su máquina.

Instalacion de dependencias:

`npm install`  

Para iniciar servidor:

`npm run dev`  

Visitar pagina:

`http://localhost:5173/`
